# Data Structures & Algorithms with C#

A practical public learning project by **Mcebo Sbonelo Zulu**.

This repository explains Data Structures and Algorithms using C# with clean examples, real-world explanations, and problem-solving practice.

## Topics Covered

- Arrays
- Linked Lists
- Stacks
- Queues
- Hash Tables
- Trees
- Graphs
- Searching Algorithms
- Sorting Algorithms
- Big O Notation
- Recursion
- Dynamic Programming

## Goal

The goal of this project is to help students, junior developers, and problem solvers understand how data structures and algorithms work using modern C#.
